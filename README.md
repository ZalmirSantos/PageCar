# PageCar

A car-themed landing page developed using pure HTML and CSS.

## 📌 About the Project

This project was created during the beginning of my web development learning journey.

The goal was to build a complete static page from scratch, inspired by modern automotive websites, using only HTML and CSS.

## 🚀 Features

- Car sales landing page
- Custom layout design
- Styled sections and content blocks
- Pure HTML and CSS structure
- Beginner front-end project

## 🛠️ Built With

- HTML5
- CSS3

## 📂 Project Structure

- `index.html` → Main page structure  
- `venda.css` → Styling and layout

## ▶️ How to Run

Open `index.html` in your browser.

## 🎯 Purpose

This project represents one of my first practical experiences building websites and applying front-end concepts.
